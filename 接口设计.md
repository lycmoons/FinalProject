- 登录POST  FromBody

http://localhost:8080/User/login......

前端发送邮箱、密码，后端返回用户token



- 注册POST   FromBody

http://localhost:8080/User/addUser

前端发送姓名、密码、邮箱



- 发送邮箱验证码POST    FromQuery

http://localhost:8080/User/SendCode

前端发送邮箱，后端对邮箱发送验证码



- 验证验证码POST    FromBody

http://localhost:8080/User/VerifyCode

前端发送email、验证码，后端返回是否匹配(通过http状态码，200表示匹配，400表示不匹配)

- 获取基于决策树的企业信用等级评估结果 **POST**

前端发送：

```json
{
    debt_rate:         // 资产负债率（%）
    interest:          // EBITDA利息数（倍）
    inventory_turn:    // 存货周转次数（次）
    flow_debt:         // 现金流动负债比率或EBITDA（%）
    cost:              // 成本费用利润率（%）
    owner_equity:      // 所有者权益合计（万元）
}
```

后端发送：

```json
{
    debt_rate: []
    interest: []
    inventory_turn: []
    flow_debt: []
    cost: []
    owner_equity: []
}
// 其中每一项都是一个有 6 个元素的数组
// 前 5 个元素分别是这一个指标在整个企业所在行业中的 最小值、Q1、中位数、Q2、最大值
// 第 6 个元素是这个企业这一指标的值
```

```json
{
    i1:     // 流动资产周转率
    i2:     // 应收账款周转率
    i3:     // 纳税信用等级
    i4:     // 1/财务费用率
    i5:     // 1/产权比率
    i6:     // 流动比率
    i7:     // 资产负债率
    i8:     // 利息保障倍数
    i9:     // 现金流量利息保障倍数
    i10:    // 营业收入增长率
    i11:    // 总资产增长率
}
```

后端发送：

```json
{
    o1:    // 流动性
    o2:    // 筹资
    o3:    // 清偿
    o4:    // 成长力
    o5:    // 财务风险综合指标
}
```
